# dailynews 폴더

daon.io.kr Daily뉴스(오늘의 뉴스한입) 카드 이미지를 보관하는 곳입니다.

## 규칙
- 파일명: `YYYY-MM-DD.png` (예: 2026-07-10.png)
- 공개 주소: `https://daon.io.kr/dailynews/YYYY-MM-DD.png`
- 매일 이 폴더에 카드 PNG 1장만 추가하면 됩니다.

## 흐름
1. 카드 PNG를 이 폴더에 올림
2. Notion 카드뉴스 DB에 행 추가 (이미지URL = 위 공개 주소, 게시상태=발행)
3. daon.io.kr Daily뉴스 블록에 자동 표시
